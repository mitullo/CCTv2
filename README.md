# Adaptive PASAT · CCT

A local-first cognitive control training app built around paced N-back arithmetic. The interval
adapts to correct and incorrect streaks while session data stays in the browser.

## Training controls

- Addition, multiplication, signed subtraction, and absolute difference
- 1-back through 5-back working-memory depth
- Configurable 1–9 number pool and optional single-digit answers
- Audio, visual, or combined stimulus presentation
- Keyboard input plus an optional mobile number pad
- Timed sessions or a fixed correct-answer goal
- Adjustable staircase step, floor, and streak thresholds
- Three error sounds, volume control, error flash, and dark mode
- Local session history, trend charts, and JSON export/import
